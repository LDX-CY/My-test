四、安装配置clash代理

``` 下载地址：https://github.com/Dreamacro/clash/releases

来到用户主目录下

cd ~

创建文件夹 clash

mkdir clash

cd clash

将刚才下载的文件拷贝到 ~/clash中后，解压

rm clash-linux-amd64-v1.6.0.gz clash # 改名是为了更方便进行后续操作

运行

./clash

配置文件修改

路径：~/.config/clash

第一次运行clash 会创建~/.config/clash 文件夹，以及其中的两个文件config.yaml 和 Country.mmdb，Country.mmdb不用管，config.yaml 是我们需要修改的配置文件，默认下载下来的config.yaml没用，需要通过wget 或 curl 从订阅地址处获取配置文件

```

```

在 ./clash 进程仍在运行的情况下，打开浏览器，输入网址：http://clash.razord.top/#/proxies 可以看到 clash 的 UI 界面。
```
